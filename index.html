Formação em Programação: do Zero Absoluto ao Flutter Profissional

Uma formação completa e progressiva para formar programadores de verdade — começando com lógica e análise de sistemas, evoluindo por backend, bancos de dados, web, deploy e finalizando com Flutter avançado e um app de alto nível.

2 aulas por semana 1h30 por aula Progressão ultra-gradativa Projetos guiados + desafios
  • Você aprende a pensar como programador: lógica, algoritmos e modelagem antes do código.
  • Constrói projetos em cada etapa — do primeiro “Hello World” até CRUDs completos, APIs e deploy.
  • Portfólio real: vários sistemas + múltiplos apps Flutter + capstone final integrado.
  • Conteúdo organizado para reduzir a curva de aprendizado e evitar lacunas.

Como Funciona

Como as aulas funcionam (1h30, 2x por semana)

Estrutura fixa da aula

  1. 10 min Check-in (recuperação): 3 perguntas rápidas para fixar o conteúdo anterior
  2. 15 min Conceito (1 ideia por vez)
  3. 15 min Exemplo resolvido (o professor “pensa alto” e mostra o caminho)
  4. 35 min Lab guiado (projeto do dia)
  5. 10 min Desafio individual (variação do que foi feito)
  6. 5 min Checklist + tarefa

Você aprende com exemplos bem construídos e pratica imediatamente. A cada semana, o conteúdo volta de forma espaçada para consolidar memória e reduzir esquecimento.

Avaliação e evolução

  • Pontuação por aula (10 pontos): check-in + lab + desafio
  • Projetos por módulo: microprojetos + projeto final
  • Rubricas objetivas: funcionalidade, validação, organização, clareza e entrega

Metodologia clara, com prática e revisão inteligente

Duracao: Semanas 1–6 (Aulas 1–12)

Aula 1: Lógica & Algoritmos (pseudocódigo, decisões, repetição, fluxogramas)

Aula 2: Análise de Sistemas (requisitos, critérios, casos de uso, wireframes)

O que aprende

  • Estruturar raciocínio com decisões e repetição.
  • Traduzir requisitos em casos de uso e wireframes.
  • Definir regras e critérios de validação.

Entregas

  • OrderHub v0 no papel: EPS, fluxograma, pseudocódigo do menu, requisitos RF/RNF, user stories, caso de uso com erros, wireframes e cenários de teste

Duracao: Semanas 7–12 (Aulas 13–24)

Aula 1: Python I (base sólida e progressiva)

Aula 2: Ferramentas do programador (terminal, Git, debug, organização)

O que aprende

  • Sintaxe e estruturas fundamentais em Python.
  • Uso de terminal, Git e boas práticas de organização.
  • Primeiro CRUD em linha de comando com validação.

Entregas

  • OrderHub CLI v1 em Python: menu + CRUD simples + validações
  • Repositório Git com README e padrão de commits

Duracao: Semanas 13–18 (Aulas 25–36)

Aula 1: Python II (dados, arquivos, relatórios)

Aula 2: Java I (OOP do zero: classes, objetos, encapsulamento)

O que aprende

  • Manipular arquivos, dados e relatórios com Python.
  • Conceitos de OOP em Java: classes, objetos e encapsulamento.
  • Modelar domínios reais com objetos.

Entregas

  • Analisador de CSV + relatórios (Python)
  • Domínio OO do OrderHub (Java)

Duracao: Semanas 19–24 (Aulas 37–48)

Aula 1: SQL (modelagem ER → SQL Postgres/MySQL → JOIN/GROUP BY)

Aula 2: Engenharia de Software I (UML, camadas, organização, testes básicos)

O que aprende

  • Modelar dados e escrever consultas SQL com JOINs e agregações.
  • Organizar camadas e documentar arquitetura com UML.
  • Criar testes básicos e validar regras.

Entregas

  • DER do OrderHub + criação de tabelas + consultas e relatórios
  • Documento de arquitetura v1 + testes do domínio

Duracao: Semanas 25–30 (Aulas 49–60)

Aula 1: HTML + CSS (estrutura → layout → responsivo → formulários)

Aula 2: JavaScript I (base → DOM → eventos → validações → LocalStorage)

O que aprende

  • Estruturar páginas e construir layouts responsivos.
  • Manipular DOM, eventos e validações em JavaScript.
  • Persistir dados locais com LocalStorage.

Entregas

  • Site institucional (responsivo)
  • Admin web (front-only) do OrderHub com lista/cadastro e persistência local

Duracao: Semanas 31–36 (Aulas 61–72)

Aula 1: JavaScript II (async/fetch/APIs)

Aula 2: Node.js I (Express, REST, CRUD + SQL)

O que aprende

  • Consumir APIs com async/await e fetch.
  • Criar API REST com Express e integração SQL.
  • Tratar erros, loading e respostas.

Entregas

  • Front consumindo API (loading/erro + CRUD)
  • API REST conectada ao banco (CRUD completo)

Duracao: Semanas 37–42 (Aulas 73–84)

Aula 1: NoSQL (MongoDB: modelagem, queries, agregações, índices)

Aula 2: Node.js II (auth/JWT, roles, segurança e padrões de erro)

O que aprende

  • Modelar dados em MongoDB e otimizar consultas.
  • Implementar autenticação JWT e controle de acesso.
  • Padronizar erros e proteger rotas.

Entregas

  • MongoDB aplicado (ex.: logs/auditoria e coleções documentais)
  • API segura com autenticação + autorização

Duracao: Semanas 43–48 (Aulas 85–96)

Aula 1: PHP (CRUD, sessões, MVC simples)

Aula 2: Deploy/DevOps (Docker, env, pipeline simples, publicação)

O que aprende

  • Construir CRUDs com PHP e MVC simples.
  • Configurar ambientes e publicar com Docker.
  • Documentar processos de deploy.

Entregas

  • Mini sistema PHP com CRUD
  • Deploy com Docker + runbook (como rodar, logs, ambiente)

Duracao: Semanas 49–54 (Aulas 97–108)

Aula 1: Dart (linguagem, OOP, async)

Aula 2: Flutter Foundations (UI, navegação, forms, estado, persistência local)

O que aprende

  • Sintaxe Dart, OOP e async.
  • Criar UI em Flutter com formulários e estado.
  • Persistir dados offline.

Entregas

  • Projetos pequenos em Dart
  • CRUD offline em Flutter (base para o avançado)

Duracao: Semanas 55–72 (Aulas 109–144)

Aulas 1 e 2: Flutter Advanced + Capstone

O que aprende

  • Arquitetura Flutter avançada e integração com API.
  • Gestão de estado, cache e offline.
  • Qualidade final, refino de UX e preparação de capstone.

Entregas

  • Série de apps progressivos (6 apps) + Capstone final “alto nível”:
  • App 1: UI Pro (catálogo, tema, componentes)
  • App 2: Estado (lista, filtros, organização)
  • App 3: Persistência robusta (offline)
  • App 4: API + cache + erros (dashboard)
  • App 5: Auth real (JWT + rotas protegidas)
  • App 6: UX avançada (animações, microinterações)
  • Capstone: App profissional integrado com API, autenticação, CRUD completo, busca/filtros, offline/caching e qualidade

Capstone Flutter

Módulo Final: Flutter Avançado + Capstone (app de alto nível)

O fechamento da formação é um projeto completo, com padrão profissional.

Funcionalidades do capstone

  • Autenticação real (JWT) + rotas protegidas
  • CRUD completo (clientes/produtos/pedidos)
  • Busca, filtros e estados vazios
  • Tratamento de erros (loading/erro/retry)
  • Offline/caching (fallback e consistência)
  • Organização em camadas (models/services/controllers)
  • Qualidade: testes básicos + README + apresentação final
